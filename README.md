💰 C++ Finance Manager (Консольний Фінансовий Менеджер)
Простий, але потужний консольний додаток для управління особистими та спільними фінансами, написаний на C++. Програма дозволяє відстежувати доходи, витрати, керувати різними типами рахунків та аналізувати свій загальний капітал у різних валютах.

✨ Головні можливості (Features)
💼 Управління рахунками: Підтримка звичайних гаманців, кредитних карток (із кредитним лімітом та власними коштами) та спільних бюджетів для кількох користувачів.

💱 Мультивалютність: Робота з UAH, USD, EUR. Можливість самостійно оновлювати курси валют та рахувати свій "Загальний капітал" в одній обраній валюті.

🔄 Перекази: Зручний переказ коштів між власними рахунками з автоматичною конвертацією валют за актуальним курсом.

📊 Аналітика та Виписки: Генерація звітів (ТОП-3 витрат, статистика по користувачах) та детальна історія транзакцій (виписка) для кожного рахунку.

🌍 Двомовний інтерфейс: Підтримка української та англійської мов із можливістю зміни "на льоту".

💾 Локальне збереження: Усі дані (рахунки, історія, баланси) автоматично зберігаються у файл finance_data.txt.
